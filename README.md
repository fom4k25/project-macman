# Проект «MacMan» (Pac-Man на Python)

## Описание проекта  
Учебный проект. Цель — создать игру «MacMan» и отработать навык использования системы контроля версий git, работы с ветками и удалённым репозиторием.

## Используемые технологии  
- Python 3.x  
- Библиотека pygame  
- Git + GitHub (или GitLab/Bitbucket)  
- Ветки: `master` (стабильная), `dev` (разработка)

## Ветки и workflow  
- `master` — стабильная версия проекта, сюда вливаются только проверенные изменения.  
- `dev` — ветка разработки: все новые функции, исправления ведутся здесь.  
- При необходимости можно создавать дополнительные ветки от `dev`, например `feature/новая-функция`.
